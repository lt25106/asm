# ASMViz

A VSCode Extension for x86 AT&T syntax assembly. Currently still in alpha.

## Features

- Syntax Highlighting
- Hover Tooltips

## Requirements

- `gcc` (Must be in your system PATH)

## Installation
1. Go to https://github.com/lt25106/asm/actions/workflows/vsix.yaml
2. Click on the newest successful workflow run
3. Scroll down to the Artifacts section
4. Download the `extension-vsix`
5. Open the Extensions view (Ctrl+Shift+X) in VSCode
6. click the ... (Views and More Actions)
7. select Install from VSIX....
